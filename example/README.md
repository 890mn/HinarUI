# Example

## Based on

- VSCode + platformIO
- ESP32-WROVER-E
- 0.96' OLED I2C 128x64 with B/Y Double Color AT
  - OLED_SDA **21**
  - OLED_SCL **22**
  - OLED_ADDR **0x3C**
- Button AT
  - KEY_BACK **35**
  - KEY_ENTER **32**
  - KEY_CYCLE **33**
- SHT30 AT
  - SHT30_SDA **18**
  - SHT30_SCL **19**
  - SHT30_ADDR **0x44**
